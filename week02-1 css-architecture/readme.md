# Responsive Design & CSS Architecture

## Overview

This project demonstrates a responsive landing page built using **HTML5** and **CSS3**. The page follows a **mobile-first approach** and adapts seamlessly to mobile, tablet, and desktop devices using media queries.

## Features

- Responsive Navigation Bar
- Hero Section with Call-to-Action Button
- Features Section
- About Section
- Contact Form
- Footer
- Mobile-First Design
- Fluid Typography using `clamp()`
- Responsive Layout using CSS Grid
- Three Responsive Breakpoints
- Clean and Reusable CSS Structure

## Responsive Breakpoints

- **Mobile:** Default layout (below 768px)
- **Tablet:** 768px to 1023px
- **laptop:** 1024px and above

## CSS Architecture

This project follows organized CSS practices, including:

- Mobile-First CSS
- Reusable Components
- CSS Grid Layout
- Media Queries
- Consistent Naming Convention
- Responsive Typography
- Flexible Layout Design

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Media Queries
- Responsive Design
- `clamp()` for Fluid Typography
